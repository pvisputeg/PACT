# Genuine GPT-5.6 artifact location

`npm run generate:agents` writes a fresh, schema-validated Operation Northstar artifact here. It must identify `northstar-material-recovery-v1`, the OpenAI Agents SDK, manager orchestration, separate Outcome Lead and Auditor roles, both response IDs, both trace IDs, token usage, and project-local cost.

The currently checked-in JSON predates Operation Northstar. It remains historical provenance and is never loaded or represented as Northstar evidence. The local judge path uses the transparent Northstar fixture until generation is intentionally run with `OPENAI_API_KEY`.

The raw paid candidate is written to `artifacts/gpt-5.6/candidate-artifact.json` before release acceptance. Only a Northstar-identified artifact that passes schema, evidence, cross-functional, independent-audit, provenance, text-quality, and budget checks is promoted.
