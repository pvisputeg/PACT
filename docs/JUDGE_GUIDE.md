# Judge guide - Operation Northstar

## Fast public path

Open the [free public judge demo](https://pvisputeg.github.io/PACT/?scenario=northstar&artifact=auto&reset=1#/). No credentials or API key are required. Confirm the visible model-artifact provenance before following the walkthrough below.

## Fast local path

1. Run `npm install` and `npm run judge:verify`.
2. Run `npm run dev` and open `http://localhost:5173/?scenario=northstar&artifact=fixture&reset=1`.
3. Confirm the interface says `LOCAL SCHEMA FIXTURE · NO API CALL` and `SYNTHETIC OPERATING ENVIRONMENT`.
4. Select **Enter Outcome Room**.
5. Move from investigation to **Define the governed outcome**.
6. Select **Accept contract and verify signal** and confirm 8.1 ERP days reconcile to 5.4 usable days.
7. Select **Explore enterprise impact** and inspect the seven-domain cascade.
8. Select **Compare bounded strategies**. Speed must fail budget, Cost must fail target, and Balanced must be the only compliant choice.
9. Select **Submit for independent audit**. Confirm a separate Auditor reports five material findings and `Decision-ready with conditions`.
10. **Bind all 5 Auditor conditions** by continuing to the Action Contract.
11. Select **Approve with conditions and activate**. This is the named Plant COO decision.
12. Confirm the exact rejection: `Action blocked: Required quality authorization is missing.`
13. Select **Run approved graph**. Confirm ten actions complete, the customer artifact remains `not_sent`, and the twin reaches 9.8 coverage days.
14. Select **Measure the outcome**, then Day 21. Confirm target 95.0%, simulation 96.4%, observed synthetic 96.1%, and final spend $389,000.
15. Select **Accept closeout**, then inspect Outcome Ledger and Replay.

## What this proves

- Models propose and challenge; they do not authorize or execute.
- Policy guards enforce proof, independent audit, human approval, quality, finance, predecessors, supplier scope, and draft-only communication.
- Target, simulation, and observation remain separate.
- The outcome does not stop at a recommendation: it closes with a measured synthetic result and reusable learning.
